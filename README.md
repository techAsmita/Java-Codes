# ☕ Java Basics

My first hands-on exploration of Java — written while picking up the language from scratch, coming from a background in C and C++.

Each file covers a distinct fundamental concept, with commented-out experiments showing the thought process as things were figured out.

---

## 📂 Files

| File | Concept |
|------|---------|
| `Main.java` | Hello World, variables, data types, basic arithmetic, `Scanner` input |
| `Main2.java` | Functions — `void` methods, return types, area of rectangle & square |
| `Main3.java` | Operators — float arithmetic, string concatenation, relational & logical operators, `Math.round()` |
| `Main4.java` | Conditional statements — `if/else if/else`, `switch/case`, calculator with `Scanner` |
| `Main5.java` | Loops — `for`, `while`, even/odd check |
| `Main6.java` | Arrays — declaration, `.length`, traversal, element update |
| `Questions.java` | Practice problems — sum & mean of array, Celsius to Fahrenheit, Nth term of GP, switch-case number-to-word |

---

## 🧠 Concepts Covered

- `System.out.println` vs `System.out.print`
- Primitive types: `int`, `float`, `boolean`, `char`, `String`
- `Scanner` for user input (`nextInt()`, `nextFloat()`, `next().charAt(0)`)
- Static methods and return types
- Arithmetic, relational (`<`, `>=`, `==`) and logical (`&&`, `||`, `!`) operators
- `if / else if / else` and `switch / case / break / default`
- `for`, `while` loops
- Arrays: declaration, `.length`, traversal, mutation
- `Math.round()` from the Java Math library
- Key difference from C/C++: `.length` instead of `sizeof`

## 🛠️ How to Run

```bash
javac Main.java
java Main
```

No external dependencies — pure Java (JDK 8+).
